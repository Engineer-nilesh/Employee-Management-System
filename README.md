This repository contains a step-by-step implementation of a complete Employee Management System built using .NET 8 Blazor WebAssembly for the frontend and a .NET 8 Web API for the backend.
The project demonstrates how to build a modern, scalable, and interactive web application using the latest .NET technologies. It covers everything from setting up the project structure to implementing real-world features commonly used in enterprise applications.

# In this project, you will learn how to:
- Build a Blazor WebAssembly (WASM) application using .NET 8
- Create a RESTful API with .NET 8
- Implement CRUD operations (Create, Read, Update, Delete) for employee data
- Connect a Blazor WASM frontend to a secure and efficient API backend
- Design a modern and responsive UI using Blazor components
- Add advanced features such as:
   - Printing employee records
   - Exporting data to PDF
- Handle data flow, validation, and error handling between client and server
- Structure a real-world application following best practices
# Tech Stack
- Frontend: Blazor WebAssembly (.NET 8)
- Backend: ASP.NET Core Web API (.NET 8)
- Database: SQL Server (or any supported database)
- ORM: Entity Framework Core
- PDF Export: (e.g., iText / QuestPDF / similar library)

# Features
- Employee listing with search and filtering
- Add, edit, view, and delete employees
- Print employee details
- Export employee data to PDF
- Clean and maintainable code structure
- Built with .NET 8, following modern development standards
